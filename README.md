# Metadata Scrubber

A fast, privacy-focused web application that removes EXIF metadata from images while preserving quality. Built with Python Flask backend and vanilla JavaScript frontend.

---

## Features

- **Lightning Fast** - Optimized processing for images up to 100MB
- **100% Private** - All processing happens locally, no cloud uploads
- **Quality Preservation** - Removes metadata without affecting image quality
- **Detailed Stats** - Processing time, file size, dimensions, and speed metrics
- **Modern UI** - Beautiful gradient design with drag-and-drop support
- **Responsive** - Works on desktop, tablet, and mobile
- **Batch Processing** - CLI tool supports multiple files
- **Format Support** - JPEG, PNG, WEBP, BMP, TIFF

### What Gets Removed

GPS Location  
Camera Model & Make  
Date & Time Taken  
Camera Settings (ISO, Aperture, Shutter Speed)  
Software Information  
Copyright & Artist Data  
Thumbnail Images  
All EXIF Tags
